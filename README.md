# CellScript iCKB Equivalence

Executable benchmark data, fixtures, original binary evidence, and reports for the CellScript iCKB equivalence track.

The parent CellScript compiler repository consumes this repository as a submodule at `tests/benchmarks`.

## Contents

- `ickb_specs/`: CellScript benchmark specs.
- `ickb_diff/`: differential matrix, claim manifest, CKB VM fixtures, and original binaries.
- `ickb_positive/`: accepted model fixtures.
- `ickb_negative/`: rejected model fixtures.
- `docs/`: carried evidence reports and branch notes.
